# VTVGO WORLD CUP 2018 HD

### Requirement
- Golang 1.10.x

### Run
- Install `vgo` (https://github.com/golang/vgo)
- Run test with command `vgo run world-cup.go`

### Usage
Open Stream URL `http://localhost:8080/live/vtv3` or `http://localhost:8080/live/vtv6` on VLC
